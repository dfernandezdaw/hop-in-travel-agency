## Run Locally

Clone the project

```bash
  git clone https://github.com/dfernandezdaw/hop-in-travel-agency
```

Go to the project directory

```bash
  cd frontend
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```

To run backend server, go to project directory

```bash
  cd backend
```
Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```

## Tech Stack

**Frontend:** React, CSS
**Backend:** NodeJS, Express, MongoDB
